Brand Voice Guidelines

Core principles for Block communications

01

Human-First Communication

  • Write like talking to a friend
  • Natural, genuine language
  • Emotionally resonant
02

Clarity Through Simplicity

  • Get to the point
  • Short sentences
  • Avoid jargon
03

Playful Professionalism

  • Balance warmth with expertise
  • Approachable but credible
  • Subtle humor where appropriate

✓ Do

  • Use contractions
    ✓ "We're launching a new feature" not "We are launching a new feature"
  • Write in active voice
    ✓ "Cash App now supports Bitcoin" not "Bitcoin is now supported by Cash App"
  • Use everyday language
    ✓ "Send money instantly" not "Facilitate instantaneous monetary transfers"
  • Be specific with numbers
    ✓ "50 million users" not "millions of users"
  • Address reader directly
    ✓ "You can now send money to anyone" not "Users can now send money"

✗ Don't

  • Use corporate buzzwords
    ✗ "Leverage synergies to optimize outcomes" → ✓ "Work together to get better results"
  • Write long, complex sentences
    ✗ "In order to facilitate the process of..." → ✓ "To help you..."
  • Use vague language
    ✗ "We're making improvements" → ✓ "We're adding instant transfers"
  • Hide behind passive voice
    ✗ "A decision was made" → ✓ "We decided"
  • Use outline-like conclusions
    ✗ "As we look to the future, challenges and opportunities await" → ✓ "Here's what's next: [specific action]"
  • Use negative parallelisms
    ✗ "It's not about the money, it's about access" → ✓ "This is about access"

⚠ Block-Specific Rules

Avoid Outline-like Conclusions

Generic wrap-ups that sound like essay conclusions. End with action, not reflection.

❌ "challenges and opportunities ahead"
❌ "as we look to the future"
❌ "moving forward"
❌ "remains to be seen"

Avoid Negative Parallelisms

Structures that define by what something is NOT. State what it IS.

❌ "It's not about X, it's about Y"
❌ "Not just X, but Y"
❌ "Less X, more Y"
❌ "Rather than X, we Y"
